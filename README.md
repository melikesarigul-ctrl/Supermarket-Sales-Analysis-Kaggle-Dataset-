# Supermarket Sales Dataset Analysis using Power BI

# Project Overview 
-Interactive Power BI Sales Dashboard with dynamic currency conversion (USD/TRY) via live National Bank of Ukraine API integration. Features time-series analysis, category distribution, and revenue modeling.

# Tools and Data Sources
-Power BI (Data Modeling & Visualization)
-CSV (Raw Data)
-NBU API: (https://bank.gov.ua/en/open-data/api-dev)
-Source: Kaggle – Supermarket Sales Dataset (https://drive.google.com/file/d/1HaM0jEJNCXBrCkbt-InJGLokmm1EUxuc/view)

# Key Analysis Performed
-Total Sales & Gross Income
-Sales by Currency & Branch
-Average of Unit Price by Product Line
-Payment Method Analysis
-Sales by Month and City
-External API Integration – National Bank of Ukraine
-Product Distribution by Category

# Key Insights
-Currency Impact: Real-time USD rates allow us to see our true revenue in different currencies.
-Top Products: "Fashion Accessories" and "Sport and Travel" items are the most popular categories.
-Payment Methods: Most customers prefer  "Ewallet" and "Cash" over Credit Card.
-Branch Comparison: Branch "C" shows the highest efficiency in overall sales volume.
-Peak Sales Month: "January" is the highest performing month in the dataset, showing a significant lead in total sales compared to other months.

# Conclusion
-This project successfully demonstrates how a static sales dataset can be transformed into a dynamic business tool. By integrating the NBU API, we moved beyond simple reporting to real-time financial analysis.
